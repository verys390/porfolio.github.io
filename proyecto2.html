STANLEY E-commerce Responsive

Proyecto académico desarrollado con Bootstrap 5, simulando una tienda online responsive para la marca Stanley.

La propuesta incluye 7 productos organizados en pastillas, diseño mobile-first, estructura semántica y componentes personalizados para crear una experiencia clara y moderna.

Sistema de Grilla

El diseño del sitio está construido sobre el sistema de grillas de 12 columnas de Bootstrap 5, asegurando una estructura flexible y adaptativa.

Se aplicaron clases como col-12, col-md-6 y col-lg-4, optimizando el contenido para mobile, tablet y desktop sin generar scroll horizontal.

Pastillas de Productos

El catálogo de productos fue desarrollado con .card de Bootstrap. Cada producto cuenta con imagen, título, descripción, precio y botón “comprar”.

Las tarjetas se distribuyen con clases col-12 en mobile, col-md-6 en tablets y col-lg-4 en escritorio, cumpliendo con los breakpoints del framework.

Se desarrollaron 7 productos únicos con coherencia visual y una estructura limpia.

Componentes y Detalles

En todo el sitio se integraron componentes de Bootstrap 5: botones personalizados, formularios, tarjetas y utilidades como márgenes, paddings, colores y sombras.

Estos elementos fueron combinados con estilos propios para lograr una interfaz equilibrada, funcional y visualmente atractiva.

Reflexión del Proyecto

Este proyecto fue una gran oportunidad para afianzar mis conocimientos en Bootstrap 5. Me encantó trabajar con esta herramienta, ya que me permitió estructurar de forma rápida y ordenada el contenido, aplicar estilos visuales consistentes y adaptar el diseño a distintos dispositivos con mucha fluidez.

Sentí una clara mejora en mi uso de Bootstrap: pude entender mejor cómo funciona el sistema de grillas, cómo combinar clases utilitarias y cómo personalizar componentes para que se ajusten a una estética propia.

Este trabajo también me ayudó a pensar el diseño desde una lógica más integral, conectando funcionalidad, visualidad y experiencia de usuario. Me entusiasma seguir profundizando en este camino, combinando creatividad con herramientas técnicas.

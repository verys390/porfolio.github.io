Descripción del Proyecto

Clínica Salud Total es una plataforma web interactiva que permite a los usuarios gestionar turnos médicos y simular planes de salud personalizados de forma rápida y eficaz.

Desarrollado con tecnologías modernas como JavaScript (ES6), Bootstrap 5 y LocalStorage, este sistema ofrece formularios dinámicos, cálculos de cotización inteligentes y una interfaz completamente responsive.

La experiencia se complementa con alertas visuales (SweetAlert2), validaciones en tiempo real y un flujo de usuario fluido.

Tecnologías utilizadas

Clínica Salud Total fue desarrollada con tecnologías modernas que garantizan una experiencia fluida, accesible y visualmente amigable para los usuarios.

  • HTML5 y CSS3: estructura semántica y diseño responsive.
  • Bootstrap 5: sistema de grillas, componentes y utilidades modernas.
  • JavaScript: lógica dinámica, eventos y manejo modular.
  • SweetAlert2: alertas visuales interactivas.
  • LocalStorage: persistencia local de turnos y contrataciones.

Funcionalidades clave

Formulario de turnos con validaciones en tiempo real

El sistema cuenta con un formulario inteligente que permite a los usuarios ingresar sus datos personales, seleccionar una especialidad médica, y elegir la fecha y el horario deseado.

Las validaciones se aplican en tiempo real mediante JavaScript, asegurando que todos los campos requeridos estén completos, y evitando la asignación de turnos duplicados mediante chequeo en LocalStorage. Esto garantiza una experiencia fluida y previene errores comunes antes de enviar la solicitud.

Simulador de planes con cálculo personalizado

El sistema incorpora un simulador dinámico de planes de salud que evalúa múltiples variables ingresadas por el usuario, como edad, zona geográfica, tipo de empleo y cantidad de personas a cubrir.

Con base en esa información, se calcula de forma automática el plan más adecuado para el perfil ingresado, mostrando una propuesta personalizada con su precio estimado. Esta lógica está implementada íntegramente en JavaScript y puede adaptarse fácilmente a reglas comerciales más complejas.

Modal de contratación con confirmación

Luego de completar la simulación de un plan de salud, el sistema presenta al usuario un modal interactivo de confirmación, desarrollado con la biblioteca SweetAlert2, que mejora la experiencia visual e impulsa la toma de decisiones.

Este modal muestra de manera clara los detalles del plan recomendado (nombre y precio) y brinda la opción de confirmar la contratación. Si el usuario acepta, la información se guarda automáticamente en LocalStorage, simulando el alta del plan sin necesidad de backend. Esta funcionalidad está pensada para integrarse fácilmente con APIs reales o sistemas CRM en futuras versiones.

Diseño mobile-friendly

Toda la interfaz fue desarrollada con un enfoque responsive-first, garantizando una correcta visualización y navegación en dispositivos móviles, tablets y pantallas de escritorio.

Gracias al uso del sistema de grillas y utilidades responsivas de Bootstrap 5, el diseño se adapta automáticamente al ancho de cada dispositivo, manteniendo la legibilidad, accesibilidad y funcionalidad sin necesidad de ajustes manuales por parte del usuario.

Gourmet Selection

Prototipo de aplicación móvil orientado a facilitar la planificación de experiencias gourmet. Combina productos seleccionados, recetas y maridajes para crear una app intuitiva y visualmente atractiva.

Desarrollado con enfoque UI/UX utilizando Figma, Illustrator y Photoshop como herramientas principales de diseño.

Funcionalidades principales

Gourmet Selection es una app diseñada para facilitar la compra de productos gourmet y la planificación de experiencias culinarias únicas. Ofrece recetas integradas, recomendaciones de maridaje y un sistema de compra práctico.

  • Catálogo de Productos

    Quesos, vinos, aceites y más con imágenes, descripciones y precios claros.

  • Recetas Integradas

    Guía paso a paso para preparar platos gourmet con productos seleccionados.

  • Recomendaciones de Maridaje

    Combinaciones ideales de vinos y quesos para maximizar la experiencia culinaria.

  • Carrito de Compras

    Compra rápida, confiable y segura con revisión de productos antes del pago.

Kit de Interfaz de Usuario

El UI Kit de Gourmet Selection fue desarrollado para garantizar una experiencia visual armónica, moderna y accesible.

Incluye una selección de componentes esenciales: botones, formularios, íconos, paleta de colores, tipografía y estilo visual que mantienen la coherencia estética en toda la app.

Esta guía visual permite una implementación rápida y eficaz de nuevas pantallas, asegurando consistencia en cada interacción del usuario.

Wireframes

El proceso de diseño incluyó el desarrollo de wireframes en tres niveles de fidelidad: baja, media y alta. Esta metodología permitió planificar de forma estratégica la estructura y funcionalidad de la app, asegurando una experiencia intuitiva y coherente.

La evolución visual muestra cómo las ideas iniciales fueron tomando forma hasta llegar a un diseño final que equilibra estética, funcionalidad y usabilidad, utilizando herramientas como Figma, Illustrator y Photoshop.

Identidad Visual

La identidad visual de la app Gourmet Selection se construyó a partir de una paleta cromática cálida y sofisticada, compuesta por tonos tierra, cremas y dorados. Esta gama fue seleccionada para transmitir elegancia, confianza y armonía, cualidades propias del mundo gourmet.

Se utilizaron tipografías contrastantes: Bodoni para los títulos, que aporta carácter y jerarquía, y Open Sans para los textos, que facilita la lectura. Los íconos fueron cuidadosamente elegidos para reforzar la experiencia de usuario desde un enfoque visual claro y coherente.

Flujos de Interacción

Tarea Principal: Compra de Producto

Este flujo guía al usuario desde la selección del producto hasta la confirmación de compra. Se priorizó la claridad en la navegación, la accesibilidad a filtros y la confirmación del pedido con datos precisos de envío y pago.

El diseño intuitivo y minimalista facilita una experiencia de compra rápida y sin fricciones.

Tarea Secundaria: Recetas y Maridajes

Este recorrido ofrece inspiración culinaria mediante recetas integradas con productos gourmet seleccionados. Además, sugiere maridajes ideales entre vinos y quesos para enriquecer la experiencia gastronómica.

Esta funcionalidad potencia el valor agregado de la app, fomentando la exploración y personalización del menú.

Manual de Marca

El manual de marca establece los lineamientos visuales y conceptuales que garantizan una identidad sólida, coherente y diferenciada para Gourmet Selection. Contempla desde el logotipo, colores y tipografía, hasta aplicaciones prácticas y mockups publicitarios.

Es una herramienta clave para mantener consistencia en la comunicación visual de la marca.
